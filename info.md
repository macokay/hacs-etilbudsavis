# eTilbudsavis

Hent ugentlige tilbud fra [eTilbudsavis.dk](https://etilbudsavis.dk) direkte i Home Assistant.

Opretter én sensor pr. søgeterme med antal aktive tilbud i nærheden — med butik, pris og datoer som attributter.

Understøtter filtrering på butik, kun dåser, pris pr. liter og søgeradius. Alle indstillinger kan redigeres efter opsætning.
